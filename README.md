# Over-The-Hill Soccer League

A Khaykin and Kane production.

Project to model and forecast results for the [Over-The-Hill Soccer League](https://www.othsl.org/).

## To start the app locally

```
cd C:\Users\Owner\Desktop\othsl
python app.py
```

Then open http://localhost:5000 in your browser.

To refresh the data (download current season + recalculate ELO):

```
python refresh.py
```
