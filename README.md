# OTHSL Match Intelligence

A Khaykin and Kane production.

Live site: **[https://davidkane9.github.io/othsl/](https://davidkane9.github.io/othsl/)**

Standings, team pages, and model-driven season outlook for the [Over-The-Hill Soccer League](https://www.othsl.org/) — a recreational adult soccer league in the Greater Boston area. Features ELO ratings, 10,000-run Monte Carlo simulations, matchweek history, and what-if scenario analysis.

## Running locally

```
pip install flask frozen-flask
python app.py
```

Then open [http://localhost:5000](http://localhost:5000).

## Refreshing data

Downloads the latest results from othsl.org and recalculates ELO ratings:

```
python refresh.py
```

## Deploying

The site is a static build generated by [Frozen-Flask](https://frozen-flask.readthedocs.io/) and served via GitHub Pages from the `docs/` folder.

Deployment is automatic — pushing to `main` triggers a GitHub Actions workflow that runs `python freeze.py` and publishes the output.

To build locally:

```
python freeze.py
```

## AI email digest

`digest.py` generates a weekly narrative digest using Claude and sends it via Resend:

```
pip install anthropic resend
set ANTHROPIC_API_KEY=sk-ant-...
set RESEND_API_KEY=re_...
python digest.py          # send email
python digest.py --preview  # print HTML without sending
```
